# Test_Creation_Posts_Music

A full-stack role-based application featuring secure authentication and authorization with JWT.

## Features

* User authentication with **Login** and **Logout**
* Passwords securely hashed using **bcrypt**
* JWT token-based authentication
* Authentication token stored in **Local Storage** and **Cookies**
* Role-based access control

### User Role

* Create and manage posts
* View and listen to music

### Artist Role

* Upload and manage music
* View user posts
* Cannot create posts
* Cannot listen to music

## Tech Highlights

* Secure password hashing with bcrypt
* JWT authentication
* Role-based authorization
* Token persistence using Local Storage and Cookies
* Protected routes based on user roles

This project demonstrates secure authentication, role-based permissions, and separate dashboards for Users and Artists, making it a solid example of modern full-stack application development.
